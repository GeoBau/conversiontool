====================================
PORTFOLIO CONVERSION TOOL
INSTALLATIONS-ANLEITUNG FÜR IT
====================================

WICHTIG: Das neue Paket enthält Fehlerbehebungen!
Bitte das aktualisierte ZIP verwenden, nicht das alte.

SYSTEMVORAUSSETZUNGEN:
-------------------------------------
1. Python 3.13 oder 3.14
   Download: https://www.python.org/downloads/

2. Node.js (aktuelle LTS-Version)
   Download: https://nodejs.org/

3. Windows 10/11 oder Windows Server


INSTALLATION - SCHRITT FÜR SCHRITT:
-------------------------------------

1. ZIP-DATEI ENTPACKEN
   - portfolio-conversion-tool.zip in gewünschtes Verzeichnis entpacken
   - Empfohlen: C:\Tools\portfolio-conversion-tool

2. DEPLOYMENT AUSFÜHREN
   - Doppelklick auf: deploy.bat

   Das Skript führt automatisch aus:
   [1/5] Python-Abhängigkeiten installieren
   [2/5] Node.js-Abhängigkeiten installieren
   [3/5] Frontend kompilieren
   [4/5] Dateien prüfen
   [5/5] Abschluss

3. BEI FEHLERN

   Falls Fehler bei Pillow auftreten:
   - PowerShell als Administrator öffnen
   - Ausführen: pip install --upgrade pip
   - Ausführen: pip install --only-binary=:all: Pillow
   - deploy.bat erneut ausführen


ANWENDUNG STARTEN:
-------------------------------------

Nach erfolgreicher Installation:

1. Backend starten:
   cd api
   python app.py

2. Browser öffnen:
   http://localhost:5000
